# H.S. Darda & Co. Website

A modern, professional website for H.S. Darda & Co. - a trusted advisory firm with over 50 years of experience serving clients across India, UAE, and global markets.

## Features

- **Modern Design**: Clean, professional layout using Tailwind CSS
- **Responsive**: Fully responsive design that works on all devices
- **Interactive Elements**: Smooth animations, hover effects, and micro-interactions
- **Performance Optimized**: Lazy loading, efficient animations, and optimized code
- **Accessibility**: Semantic HTML5 structure with proper ARIA labels
- **SEO Friendly**: Clean URL structure, meta tags, and semantic markup

## Technologies Used

- **HTML5**: Semantic markup for better structure and SEO
- **Tailwind CSS**: Utility-first CSS framework for modern styling
- **JavaScript (ES6+)**: Modern JavaScript for interactions and animations
- **AOS (Animate On Scroll)**: Scroll-triggered animations
- **Font Awesome**: Professional icons throughout the site
- **Swiper**: Carousel/slider functionality (ready for implementation)

## Sections

1. **Navigation Bar**: Sticky navigation with mobile menu
2. **Hero Banner**: Compelling headline with call-to-action buttons
3. **Trust Metrics**: Key statistics showcasing experience and reach
4. **Who We Are**: Company history and core values
5. **Core Specializations**: Service offerings with detailed cards
6. **Who We Serve**: Client types and tailored solutions
7. **Client Impact & Trust**: Testimonials and success metrics
8. **Insights & Updates**: Latest articles and newsletter signup
9. **Careers**: Job opportunities and company culture
10. **Global Presence**: Office locations and target regions
11. **Contact Strip**: Easy ways to get in touch
12. **Footer**: Comprehensive footer with all important links

## Interactive Features

- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Counter Animation**: Animated counters for statistics
- **Particle Effects**: Subtle particle animation in hero section
- **Scroll Progress**: Visual progress bar showing page scroll
- **Back to Top**: Floating button for easy navigation
- **Form Validation**: Email validation for newsletter and contact forms
- **Hover Effects**: Interactive hover states on buttons and cards
- **Lazy Loading**: Images load as they come into view
- **Notification System**: User-friendly notifications for form submissions

## Getting Started

1. Open `index.html` in your web browser
2. The website is ready to use immediately
3. All dependencies are loaded from CDNs

## Customization

### Colors
The primary color scheme uses blue variants. You can customize colors by modifying the Tailwind CSS classes in the HTML file.

### Content
Update the text content, images, and links directly in the HTML file. Image placeholders use Picsum Photos - replace with actual images.

### Animations
Adjust animation speeds and effects in the `styles.css` file or modify AOS settings in `script.js`.
